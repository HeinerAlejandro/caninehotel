Aqui usaremos Unitest, para hacer pruebas respecto a los metodos del crud, es decir, si agregas una operacion rpc en algun modulo, crearas un archivo o la estructura que quieras y haras la prueba. Propuestas:

tests
	> dog.add.test.py
	> dog.modify.test.py
	> ...
	
	> room.add.test.py
	> ...

cada archivo representa una prueba
